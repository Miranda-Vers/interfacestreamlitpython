# 🏢 PCF - Portaria de Controle Facilitado
    PCF é uma solução de vanguarda para o controle de acesso predial, utilizando Visão Computacional e Inteligência de Dados para otimizar o fluxo de pessoas em ambientes corporativos e residenciais.

# 🎯 O Problema e a Solução
Portarias tradicionais sofrem com filas e falhas de identificação. O PCF automatiza esse processo, diferenciando inteligentemente os perfis de acesso:

# 👥 Convívio Normal: Moradores, proprietários e funcionários fixos possuem acesso ultra rápido via biometria facial.

# 🛠️ Especializados: Prestadores de serviço (manutenção, entregas, reformas) passam por um fluxo de autorização específica, garantindo maior rastro de segurança.

# 🚀 Funcionalidades Principais
[x] Simulador Biométrico: Interface para teste de reconhecimento de rostos e classificação de perfis.

[x] Dashboard de Tráfego: Gráficos em tempo real que mostram a densidade de acessos por categoria.

[x] Logs Técnicos: Visualização detalhada de dados para administradores e síndicos.

[x] Interface Responsiva: Design moderno e adaptável focado na experiência do operador.

# 🛠️ Tecnologias Utilizadas
O projeto foi construído utilizando o que há de mais moderno no ecossistema Python:

Streamlit: Para a criação da interface web interativa.

Pandas: Manipulação e estruturação dos dados de acesso.

Numpy: Geração de dados sintéticos para simulação de tráfego.
